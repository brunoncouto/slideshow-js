Trata-se de um projeto que utiliza HTML, CSS e JavaScript para realizar uma
apresentação de um determinado numero de imagens, respeitando um numero x
de segundos de intervalo entre cada imagem.

A apresentação é feito em um loop infinito, onde as imagens são apresentadas ininterruptamente.


EN:
This is a project that uses HTML, CSS, and JavaScript to display
a specific number of images, with a fixed interval of x
seconds between each image.

The display runs in an infinite loop, where the images are shown continuously.
